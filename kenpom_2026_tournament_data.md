## NOTES FOR LOG5 FORMULA USAGE

The Log5 formula requires win probability inputs. With KenPom data, the standard approach is:

1. Use AdjEM (= AdjO - AdjD) for each team
2. Expected margin = (Team_A_AdjEM - Team_B_AdjEM) / 2 (on a neutral court)
3. Win probability via logistic regression on the expected margin

Alternatively, using W-L records directly in Log5:
- Win% = Wins / (Wins + Losses)
- Log5(A beats B) = (pA - pA*pB) / (pA + pB - 2*pA*pB)

Where pA and pB are each team's win percentages.
